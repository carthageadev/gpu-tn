# GPU.tn

Cloud GPU rental and managed LLM endpoints for Gemma 4 and Qwen 3.6. Single-page landing built with React, TypeScript, Tailwind CSS, Three.js, and Framer Motion.

Built as a single-file Vite build using `vite-plugin-singlefile`.

## Stack

- **React 19** + **TypeScript**
- **Tailwind CSS 4** via `@tailwindcss/vite`
- **Framer Motion** — scroll-driven and viewport-triggered animations
- **Three.js** / **@react-three/fiber** — (available, used in scenes)
- **Vite 7** — dev server, build, single-file bundling

## Commands

```bash
npm run dev      # Start dev server
npm run build    # Build into a single HTML file in dist/
npm run preview  # Preview the production build
```
